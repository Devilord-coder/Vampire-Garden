# «Vampire Garden»
#### Авторы проекта
* Путилин Даниил Ильич
* Тюлина Татьяна Максимовна
## Идея
Написать 2D игру про поместье вампира.
## Задачи проекта
Задачи, которые требуются для реализации проекта:
* Изучить библиотеку Arcade;
* Спроектировать архитектуру взаимодействия;
* Разработать главное меню;
* Разработать систему сохранения игры;
* Создать систему авторизации (регистрации).
## Описание работы проекта
Первым делом пользователь регистрируется. При первой регистрации пользователь может прочитать историю главного героя. Затем появляется главное окно.
Потом можно создать новую игру или загрузить старую. В начале у пользователя есть только одно поместье. В ходе игры пользователь может развивать свои земли, а также разорять другие поместья с помощью войнов.
## Актуальность
1. Сейчас многие люди не умеют распоряжаться своими финансами. Наше приложение помогает развить навыки финансовой грамотности в игровой форме;
2. Наша игра будет помогать пользователям научиться думать наперёд, следить за последствиями своих действий.
